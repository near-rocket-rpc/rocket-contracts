# Rocket Smart Contracts

This repo contains two NEAR smart contracts of Rocket RPC:
- RPC Token Contract
  - `contracts/rpc-token`
  - testnet deployment: `token.rocket0.testnet`
- Escrow Contract
  - `contracts/rpc-escrow`
  - testnet deployment: `escrow.rocket0.testnet`
  
  ## Build
  `> make all`
